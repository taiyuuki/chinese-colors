# Chinese Colors

VS Code颜色提示扩展，适用于前端开发，可用于`html`、`css`、`less`、`sass`、`JavaScript`、`typescript`、`vue`、`react`等。

## 使用方法

* 输入<font color=HotPink>`# + 颜色`</font>即可列出预设的颜色，如果输入后没有反应请尝试 <font color=HotPink>`Ctrl + i`</font> 。
  * 支持汉字和拼音。例如：输入<font color=HotPink>`#红`</font>或<font color=HotPink>`#hong`</font>，就会列出所有与红色相关的颜色，由于是模糊匹配，使用拼音不保证100%精确。
  * 不支持英文，原因是这些颜色没有对应的英文名称。
* 颜色代码默认采用16进制格式，如果想使用RGB格式，可以在设置面板中搜索“RGB”，勾选“控制预设的中国色采用RGB格式”。
