# Chinese Colors

VS Code颜色提示扩展，可用于`html`、`css`、`less`、`scss`、`stylus`、`JavaScript`、`typescript`、`vue`、`react`等。

## 使用方法

* 输入<font color=HotPink>`# + 颜色名`</font>即可列出预设的颜色，如果输入后没有反应请尝试 <font color=HotPink>`Ctrl + i`</font> 。

  * 支持汉字、拼音。例如：输入<font color=HotPink>`#红`</font>或<font color=HotPink>`#hong`</font>，就会列出所有与红色相关的颜色，由于是模糊匹配，使用拼音时不保证100%精确。

  * 不支持英文，原因是这些颜色没有对应的英文名称。

* 颜色代码默认采用`16进制`格式，如果想使用RGB格式，可以在设置面板中搜索“RGB”，勾选“控制预设的中国色采用RGB格式”。

* 游标悬停到颜色代码上可以提示颜色名称：

  * 仅限不带透明通道的16进制格式和rgb格式。

  * 如果使用取色面板转换颜色格式，由于存在计算误差，可能会失去提示。
